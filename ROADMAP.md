# 🚀 '홀리온 (HolyOn)' 서비스 개발 로드맵 (Roadmap)

효율적인 개발을 위한 단계별 실행 계획입니다.

---

## 🛠️ 1단계: MVP 개발 (최소 가동 모델) - *현재 진행 중*
- [x] 서비스 컨셉 및 랜딩 페이지 디자인
- [x] 데이터베이스 스키마 및 API 구조 설계
- [ ] 사용자 프로필 등록 (43가지 질문 입력 폼 구현)
- [ ] 정적 매칭 데이터 시뮬레이션

## 🏗️ 2단계: 백엔드 및 실시간 기능
- [ ] 인증 서버 (Firebase/Supabase 등 간편 인증 권장)
- [ ] 하루 두 번 매칭 알림 (Push 알림 기능)
- [ ] 1:1 실시간 채팅 (Socket.io 또는 Firebase 사용)

## 🛡️ 3단계: 신뢰 및 안전 강화
- [ ] 휴대폰 본인 인증 시스템 연동
- [ ] 교단 기반 이단/사이비 원천 차단 알고리즘 구현
- [ ] 관리자 대시보드 (신고 처리 및 모니터링용)

## 🌟 4단계: 고도화 및 출시
- [ ] 사용자 답변 데이터 기반 고도화된 매칭 알고리즘
- [ ] 앱 스토어/플레이 스토어 출시 (React Native 또는 Flutter 권장)
- [ ] 교계 매체 광고 및 커뮤니티 홍보

---

## 💡 제안
현재 환경에서 **Node.js/npm**이 설치되지 않은 경우, **Firebase**나 **Supabase**와 같은 서버리스(Serverless) 플랫폼을 사용하여 별도의 서버 구축 없이 빠르게 다음 단계로 나아갈 수 있습니다. 원하신다면 이 플랫폼들을 활용한 구체적인 안내를 드릴까요?
